# PECL v0.2.9

## Overview

PECL v0.2.9 is now available.

This release focuses on fixing the mirror selection behavior after Java source discovery and latency testing.

## Highlights

- Fixed the source selection issue after using "Discover and test latency" on the Java runtime page.
- PECL now correctly follows the measured mirror result instead of keeping a slower usable source selected.

## Packages

- Windows installer (`PECL_0.2.9_x64-setup.exe`)
- Update signature (`PECL_0.2.9_x64-setup.exe.sig`)

## Installation Notes

- Close PECL before running the new installer.
- If Windows SmartScreen appears, verify the source and continue if you trust this release.
- This public repository is used for release distribution only. The private development source code is not published here.
