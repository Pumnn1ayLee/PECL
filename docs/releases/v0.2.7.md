# PECL v0.2.7

## Overview

PECL v0.2.7 is now available.

This release focuses on polish, privacy, and update-flow reliability.

## Highlights

- Fixed the app update progress rendering when the updater does not report a total package size.
- Improved the published community server list so owner names render correctly.
- Removed direct community cloud endpoint exposure from community error surfaces.
- Refined the community unavailable messaging for a cleaner user-facing experience.

## Packages

- Windows installer (`PECL_0.2.7_x64-setup.exe`)
- Update signature (`PECL_0.2.7_x64-setup.exe.sig`)

## Installation Notes

- Close PECL before running the new installer.
- If Windows SmartScreen appears, verify the source and continue if you trust this release.
- This public repository is used for release distribution only. The private development source code is not published here.
