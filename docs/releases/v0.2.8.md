# PECL v0.2.8

## Overview

PECL v0.2.8 is now available.

This release focuses on a small but important first-run UX fix.

## Highlights

- Fixed the first-run download path hint dialog so "I know" closes it correctly.
- Removed the extra default `Dialog` label from the top-left corner of that prompt.

## Packages

- Windows installer (`PECL_0.2.8_x64-setup.exe`)
- Update signature (`PECL_0.2.8_x64-setup.exe.sig`)

## Installation Notes

- Close PECL before running the new installer.
- If Windows SmartScreen appears, verify the source and continue if you trust this release.
- This public repository is used for release distribution only. The private development source code is not published here.
