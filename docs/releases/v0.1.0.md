# PECL v0.1.0

## Overview

PECL v0.1.0 is the first public release of the launcher.

This release focuses on:

- the core desktop launcher experience
- Minecraft version installation and management
- a cleaner and more stable foundation for future updates

## What's New

- Initial public Windows desktop release
- Minecraft version installation and management workflow
- Support for release and snapshot version workflows
- Loader-related workflow support for Forge, Fabric, and OptiFine
- Built-in Java runtime management inside the launcher

## Improvements

- Ongoing refinement of Minecraft installation flow
- Better task feedback, progress presentation, and cancellation handling
- General polish across launcher workflow and day-to-day usability

## Fixes

- Stability work across version installation and state synchronization
- Fixes around installation progress behavior and cancellation flow
- Improvements to launcher-side task and page linkage behavior

## Known Limitations

- The planned public server list is not included in this release
- Current release focus is Windows desktop
- This is an early public release and will continue to be refined

## Packages

- Windows installer (`Setup.exe`)
- Windows installer (`MSI`)

## Installation Notes

- Download the latest installer package from this Release page
- If you are upgrading from an older test build, close PECL before installing the new version
- If Windows SmartScreen appears, verify the source and continue if you trust this release

## Feedback
