## PECL 是什么

PECL 是一个面向 Windows 的 Minecraft 启动器，同时也是资源管理工作台。它不是只负责“启动游戏”，而是希望把玩家常用的几条流程收拢到一个更清晰、更稳定的桌面体验里：

`选择实例 -> 搜索资源 -> 安装 -> 启动游戏 -> 管理与调整`

如果你经常在 Minecraft 版本、Java、Mods、整合包、资源包、光影包、数据包之间来回切换，PECL 的目标就是让这些工作少一点分散、多一点确定性。

## 为什么还要做一个Minecraft启动器

我当然知道市面上已经有了很多数量的Minecraft启动器，一定要说为什么要做，那就是填以前脑补的坑，但更多的是兴趣使然。

## 核心能力

- Minecraft 版本安装与实例管理
- Java 检测、托管下载、自动选择、手动指定与外部 Java 隐藏/恢复
- Forge、Fabric、OptiFine 工作流支持
- Mod、整合包、资源包、光影包、数据包的浏览、下载与管理
- 资源隔离、任务进度同步、安装反馈与更新链路
- 面向社区内容浏览与后续扩展的能力基础

## 产品方向

PECL 更接近一个“Minecraft 启动与资源管理工作台”，而不是只负责启动某个版本的单一启动器。它目前持续打磨的重点主要在：

- 更稳的安装与更新体验
- 更清楚的实例与资源边界
- 更顺手的资源浏览与下载流程
- 更可控的 Java 管理与选择逻辑
